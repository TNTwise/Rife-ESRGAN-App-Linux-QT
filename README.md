<strong>Rife ESRGAN APP Linux QT</strong> Is a redesign (and hopefully an upgrade) from the original Rife ESRGAN App Linux. This program provides easy access to frame interpolation and upscaling on Linux, similar to <a target="_tab" rel="noopener noreferrer" href="https://nmkd.itch.io/flowframes">Flowframes</a> or <a target="_blank" rel="noopener noreferrer" href="https://github.com/mafiosnik777/enhancr">enhancr</a> on windows.
